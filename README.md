# sdm-runner-conteiner

Conteiner para rodar a geração de modelos de SDM e gerar as predições em um conteiner.

Os modelos são gerados com um .csv para cada espécie.

Os predicts são gerados com o modelo e as variáveis.

Variáveis utilizadas:

e.nc
tp.nc
t2m.nc
ocup_solo.nc (o mesmo arquivo para todas as previsoes)
sdtb_sul_expandido_SUL.nct2m.nc (o mesmo arquivo para todas as previsoes)
altitude.nc (o mesmo arquivo para todas as previsoes)

Arquivos de saída (exemplos):

guariba_2050.tif
guariba_2100.tif
guariba_abril_2025.tif
nigritus_2050.tif
nigritus_2100.tif
